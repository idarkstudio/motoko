# Índice

- [Array](Array.md) Proporciona funciones de utilidad extendidas para Arrays.
- [AssocList](AssocList.md) Mapa implementado como una lista enlazada de pares
  clave-valor ("Asociaciones").
- [Blob](Blob.md) Módulo para trabajar con Blobs: secuencia inmutable de bytes.
- [Bool](Bool.md) Tipo booleano y operaciones.
- [Buffer](Buffer.md) La clase `Buffer<X>` proporciona una lista mutable de
  elementos de tipo `X`.
- [CertifiedData](CertifiedData.md) Datos certificados.
- [Char](Char.md) Caracteres.
- [Debug](Debug.md) Funciones de utilidad para depuración.
- [Deque](Deque.md) Cola de doble extremo (deque) de un tipo de elemento
  genérico `T`.
- [Error](Error.md) Valores de error e inspección.
- [ExperimentalCycles](ExperimentalCycles.md) Gestión de ciclos dentro de
  actores en Internet Computer (IC).
- [ExperimentalInternetComputer](ExperimentalInternetComputer.md) Interfaz de
  bajo nivel para Internet Computer.
- [ExperimentalStableMemory](ExperimentalStableMemory.md) Acceso a nivel de
  bytes a la _memoria estable_ (virtual).
- [Float](Float.md) Números de punto flotante de doble precisión (64 bits) en
  representación IEEE 754.
- [Func](Func.md) Funciones sobre funciones, creando funciones a partir de
  entradas más simples.
- [Hash](Hash.md) Valores de hash.
- [HashMap](HashMap.md) La clase `HashMap<K, V>` proporciona un mapa hash de
  claves de tipo `K` a valores de tipo `V`.
- [Heap](Heap.md) La clase `Heap<X>` proporciona una cola de prioridad de
  elementos de tipo `X`.
- [Int](Int.md) Números enteros con signo con precisión infinita (también
  llamados enteros grandes).
- [Int16](Int16.md) Proporciona funciones de utilidad para enteros de 16 bits
  con signo.
- [Int32](Int32.md) Proporciona funciones de utilidad para enteros de 32 bits
  con signo.
- [Int64](Int64.md) Proporciona funciones de utilidad para enteros de 64 bits
  con signo.
- [Int8](Int8.md) Proporciona funciones de utilidad para enteros de 8 bits con
  signo.
- [Iter](Iter.md) Iteradores.
- [IterType](IterType.md) El tipo Iterator.
- [List](List.md) Listas simplemente enlazadas puramente funcionales.
- [Nat](Nat.md) Números naturales con precisión infinita.
- [Nat16](Nat16.md) Proporciona funciones de utilidad para enteros de 16 bits
  sin signo.
- [Nat32](Nat32.md) Proporciona funciones de utilidad para enteros de 32 bits
  sin signo.
- [Nat64](Nat64.md) Proporciona funciones de utilidad para enteros de 64 bits
  sin signo.
- [Nat8](Nat8.md) Proporciona funciones de utilidad para enteros de 8 bits sin
  signo.
- [None](None.md) El valor ausente.
- [Option](Option.md) Nulos seguros en tipos.
- [Order](Order.md) Orden.
- [OrderedMap](OrderedMap.md) Mapa clave-valor estable implementado como un
  árbol rojo-negro con nodos que almacenan pares clave-valor.
- [OrderedSet](OrderedSet.md) Conjunto ordenado estable implementado como un
  árbol rojo-negro.
- [Prelude](Prelude.md) Utilidades generales.
- [Principal](Principal.md) Módulo para interactuar con Principals (usuarios y
  canisters).
- [RBTree](RBTree.md) Mapa clave-valor implementado como un árbol rojo-negro
  (RBTree) con nodos que almacenan pares clave-valor.
- [Random](Random.md) Un módulo para obtener aleatoriedad en Internet Computer
  (IC).
- [Region](Region.md) Acceso a nivel de bytes a _regiones_ de memoria estable
  aisladas (virtuales).
- [Result](Result.md) Manejo de errores con el tipo Result.
- [Stack](Stack.md) La clase `Stack<X>` proporciona una pila LIFO mínima de
  elementos de tipo `X`.
- [Text](Text.md) Funciones de utilidad para valores de `Text`.
- [Time](Time.md) Tiempo del sistema.
- [Timer](Timer.md) Temporizadores para tareas únicas o periódicas. Aplicable
  como parte del mecanismo predeterminado.
- [Trie](Trie.md) Mapas clave-valor funcionales.
- [TrieMap](TrieMap.md) La clase `TrieMap<K, V>` proporciona un mapa de claves
  de tipo `K` a valores de tipo `V`.
- [TrieSet](TrieSet.md) Conjunto funcional.
