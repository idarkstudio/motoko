# HTTP

## Paquetes Mops para HTTP

[Mops](https://mops.one/) es un administrador de paquetes en cadena para Motoko.
Aquí hay algunos paquetes Mops para HTTP y funcionalidades web:

- [`assets`](https://mops.one/assets): Una biblioteca para agregar funcionalidad
  de canister de activos a tu canister.

- [`assets-api`](https://mops.one/assets-api): Paquete de API para canisters de
  activos.

- [`certified-assets`](https://mops.one/certified-assets): Certifica activos
  servidos a través de HTTP, asegurando la seguridad de las llamadas de consulta
  en ICP.

- [`certified-cache`](https://mops.one/certified-cache): Una interfaz única que
  almacena pares clave-valor y certifica sus hashes para ser utilizados como
  variables o activos certificados.

- [`certified-http`](https://mops.one/certified-http): Similar a
  `certified-cache`, una interfaz que almacena pares clave-valor y certifica sus
  hashes para ser utilizados como activos o variables certificados.

- [`http-loopback`](https://mops.one/http-loopback): Llama a canisters
  utilizando llamadas HTTP externas.

- [`http-parser`](https://mops.one/http-parser): Analizador de solicitudes HTTP
  para analizar URL, consultas de búsqueda, encabezados y datos de formulario.

- [`http-types`](https://mops.one/http-types): Tipos de interfaz HTTP de
  canister utilizados en `http_request` y `http_request_update`.

- [`ic-assets`](https://mops.one/ic-assets): Canister de activos con
  certificación v2.

- [`ic-certification`](https://mops.one/ic-certification): Firmas y
  certificación de canisters.

- [`ic-websocket-cdk`](https://mops.one/ic-websocket-cdk): Websockets en ICP.

- [`idempotency-keys`](https://mops.one/idempotency-keys): Paquete para generar
  UUID.

- [`motoko-certified-assets`](https://mops.one/motoko-certified-assets): Activos
  certificados de ICP.

- [`promtracker`](https://mops.one/promtracker): Seguimiento de valores de
  Prometheus.

- [`server`](https://mops.one/server): Un servidor para Motoko similar a
  Express.

- [`web-api`](https://mops.one/web-api) y [`web-io`](https://mops.one/web-io):
  Crea solicitudes HTTP y maneja respuestas.
